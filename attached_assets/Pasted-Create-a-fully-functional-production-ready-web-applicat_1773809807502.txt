Create a fully functional, production-ready web application optimized for mobile devices (mobile-first design). The application should have a modern, clean UI similar to native mobile apps.

IMPORTANT:
- The entire user interface (UI text, labels, buttons, placeholders, messages) MUST be in Uzbek language.
- Do NOT use English or Russian in the UI.

---

### 🧩 Tech Stack:
- React or Next.js
- Tailwind CSS
- Component-based architecture
- Yandex Maps API integration
- Local state management or mock backend (localStorage)

---

### 📱 Bottom Navigation (Fixed)
Create a fixed bottom navigation bar with 3 tabs:

1. Map (Xarita)
2. Problems (Muammolar)
3. Add (Muammo qo‘shish)

---

### 🗺️ Tab 1: Map (Xarita)
- Display Yandex Map centered on user's location
- User can tap anywhere on the map

On tap:
- Show a selector modal with:
  - 🔴 "Qizil (Shoshilinch)"
  - 🟡 "Sariq (Oddiy)"
- Save coordinates and selected type
- Automatically redirect user to Tab 3 (Muammo qo‘shish)

Map behavior:
- Display all saved problems as markers
- Marker colors:
  - Red → urgent
  - Yellow → normal
- On marker click:
  - Redirect to problem details page

---

### 📝 Tab 3: Add Problem (Muammo qo‘shish)
Form fields (ALL in Uzbek):

- "Muammo nomi" (text input)
- "Tavsif" (textarea)
- "Rasm yuklash" (image upload)
- "Muhimlik":
  - 🔴 "Yuqori"
  - 🟡 "O‘rta"
- "Taxminiy narx" (number input)

Button:
- "Tasdiqlash"

On submit:
- Save data (localStorage or mock API)
- Add marker to map with correct color
- Redirect user back to Map

---

### 📄 Problem Details Page (Muammo tafsilotlari)
Display:

- Muammo nomi
- Tavsif
- Rasm
- Muhimlik (rang bilan)
- Taxminiy narx

Actions:
- 💰 "Donat qilish" (simulate payment)
- 🔗 "Ulashish" (share functionality)

---

### 📋 Tab 2: Problems List (Muammolar)
- Show all problems as scrollable cards
- Infinite vertical scroll

Each card:
- Rasm
- Muammo nomi
- Qisqa tavsif
- Muhimlik rangi

On click:
- Open problem details page
